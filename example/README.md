# Braintree Document Theme Example Site

This directory contains an example site demonstrating the Braintree Document Theme.

## Running the Example

1. Navigate to this directory:
   ```bash
   cd example
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Serve the site:
   ```bash
   bundle exec jekyll serve
   ```

4. Open your browser to `http://localhost:4000`

## Structure

- `_config.yml` - Site configuration showing theme usage
- `index.md` - Homepage
- `Content.md`, `Page1.md`, `WhatsNew.md` - Example content pages
- `FeatureRequest.html`, `TeamsChat.html`, `TeamsLink.html` - Example HTML pages
- `Resources/` - Example resource pages (Downloads, Videos)

## Customization

Edit `_config.yml` to customize site settings, or add your own content files to see how they work with the theme.
