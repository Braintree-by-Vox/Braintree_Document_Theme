### Features

#### Clean Design
The theme provides a clean and professional look suitable for technical documentation.

#### Easy Customization
All settings can be configured through the `_config.yml` file.

#### Mobile Responsive
The theme is fully responsive and works on all device sizes.

#### Built on Jekyll
Uses Jekyll static site generator, which is natively supported by GitHub Pages.

## Usage Examples

### Creating a New Documentation Page

1. Create a new `.md` file in the `_docs` directory
2. Add front matter with layout and title
3. Write your content in Markdown
4. Update the navigation in `_config.yml` if needed

### Markdown Support

The theme supports all standard Markdown features:

- **Bold text**
- *Italic text*
- `Code snippets`
- Links
- Lists
- Code blocks
- And more!
